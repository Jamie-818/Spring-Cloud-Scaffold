# Spring-Cloud-Scaffold (SpringCloud脚手架)

---

 - 该项目主要提供 SpringCloud 微服务的搭建脚手架，便于大家可以快速的搭建一个简单的微服务架构。
 
---

## 该项目暂时只包含以下组件

- Eureka 注册中心(spring-cloud-eureka)
- Config 配置中心
  - spring-cloud-config
  - spring-cloud-config-client
- Ribbon 负载均衡
  - spring-cloud-ribbon-server
  - spring-cloud-ribbon-client
